# 管理员角色配置

> 自动匹配条件：用户 id 包含"admin" 或 roles 包含管理员权限、或无法匹配到其他角色时的兜底

## 核心关注
- **系统配置**：字段配置、视图管理、模块权限
- **全量数据**：所有模块的完整数据查看权限
- **数据质量**：数据完整性、重复检测、异常数据
- **用户管理**：部门结构、人员权限、角色分配
- **业务审计**：任意模块查看、跨部门数据

## 默认查询偏好
| 场景 | 推荐命令 |
|------|---------|
| 查看字段配置 | `raw GET /settings/fields?module={module}` |
| 组织架构 | `crm org` |
| 用户列表 | `crm members '{"departmentIds":[],"current":1,"pageSize":100}'` |
| 全量线索 | `crm page lead` |
| 全量商机 | `crm page opportunity` |
| 全量客户 | `crm page account` |
| 全量合同 | `crm page contract` |
| 任意模块详情 | `crm get {module} {id}` |

## 交互模式
- **默认输出**：完整信息展示，支持任意模块切换
- **数据深度**：无限制，全量数据查看
- **提醒风格**：关注系统维护和数据结构
- **行动建议**：配置优化建议、数据清理建议、权限调整建议

## 异常预警
| 场景 | 预警提示 |
|------|---------|
| 数据量异常波动 | `📊 本周线索新增量较上周下降 {N}%，请检查数据源是否正常` |
| 字段配置变化 | `ℹ️ 检测到 {module} 模块字段配置有更新` |
| 用户权限变更 | `ℹ️ 近期有 {N} 个用户权限发生变更` |
