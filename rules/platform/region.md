## 🔧 使用示例

### 查询东区线索

```bash
cordys crm page lead '{"filters":[{"name":"1751888184000015","operator":"IN","value":["东区"]}]}'
```

### 查询北区 + 南区商机

```bash
cordys crm page opportunity '{"filters":[{"name":"1751888184000030","operator":"IN","value":["北区","南区"]}]}'
```

## 📝 自定义区域映射

如果公司有自己的区域划分（如华北/华东/华南），可在公司规则中覆盖：

**位置：** `rules/company/region-custom.md`

**示例：**
```markdown
# 公司自定义区域映射

| 省份 | 自定义区域 |
|------|-----------|
| 北京 | 华北一区 |
| 上海 | 华东一区 |
| 广东 | 华南一区 |
```

**优先级：** 公司规则 > 平台规则
